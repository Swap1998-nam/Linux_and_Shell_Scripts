 Linux & Shell Scripting Cheat Sheet

This repository contains pro-level Linux commands, shell scripting examples, and DevOps-related CLI commands for quick reference and practice.

## Structure

- **commands/**: Categorized Linux/DevOps command lists
- **scripts/**: Example shell scripts
- **cheatsheet.pdf**: One-page printable cheat sheet (generated from commands)

## Usage

1. Browse commands in `commands/` for reference.
2. Try out scripts in `scripts/`.
3. Keep the PDF handy for quick reference.
